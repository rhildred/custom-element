# markdown2html
## converts markdown to html

```
<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Markdown 2 Html</title>
  </head>
  <body>
    <x-markdown2html>
# This is title
## tagline
markdown text     
    </x-markdown2html>
    <script src="https://rhildred.github.io/custom-element/custom-element.js"></script>
  </body>
</html>


```
